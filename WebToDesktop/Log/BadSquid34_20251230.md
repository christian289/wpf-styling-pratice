## 컴파일 에러 및 수정

### 에러 1: StackPanel.Spacing 속성 미지원

**에러 내용:**
```
error MC3072: XML 네임스페이스 'http://schemas.microsoft.com/winfx/2006/xaml/presentation'에 'Spacing' 속성이 없습니다.
```

**원인:**
WPF의 StackPanel은 AvaloniaUI와 달리 `Spacing` 속성을 지원하지 않습니다.

**수정 방법:**
각 자식 요소에 `Margin` 속성을 개별적으로 적용하여 간격을 구현합니다.

```xml
<!-- 수정 전 (AvaloniaUI 스타일) -->
<StackPanel Spacing="20">
    <controls:BadSquid34 />
</StackPanel>

<!-- 수정 후 (WPF 호환) -->
<StackPanel>
    <controls:BadSquid34 Margin="0,0,0,20" />
</StackPanel>
```

## Runtime Error 가능성

### 1. CloseCommand 바인딩 미설정 시
- **증상:** 닫기 버튼 클릭 시 아무 동작 없음
- **원인:** `CloseCommand` 속성이 바인딩되지 않은 경우
- **해결:** MVVM 패턴으로 Command 바인딩 또는 이벤트 핸들러 추가 필요

### 2. PART_CloseButton 이벤트 해제 미구현
- **증상:** 메모리 누수 가능성
- **원인:** `OnApplyTemplate`에서 등록한 이벤트 핸들러가 해제되지 않음
- **해결:** 컨트롤 언로드 시 이벤트 핸들러 해제 로직 추가 권장

## CSS → WPF 변환 매핑

| CSS 속성 | WPF 구현 |
|---------|---------|
| `display: flex; flex-direction: row` | `Grid` with `ColumnDefinitions` |
| `align-items: center` | `VerticalAlignment="Center"` |
| `justify-content: start` | 기본 정렬 (추가 설정 불필요) |
| `margin-left: auto` | `Grid.Column` 마지막 열에 배치 |
| `background: #EF665B` | `SolidColorBrush` 리소스 |
| `border-radius: 8px` | `CornerRadius="8"` |
| `box-shadow: 0px 0px 5px -3px #111` | `DropShadowEffect` (BlurRadius 조정) |
| `transform: translateY(-2px)` | `TranslateTransform Y="-2"` |
| `cursor: pointer` | `Cursor="Hand"` |
| `font-weight: 500` | `FontWeight="Medium"` |
