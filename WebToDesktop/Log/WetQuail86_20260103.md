# WetQuail86 변환 로그 (2026-01-03)

## 컴파일 에러 및 수정 사항

### 에러 1: CharacterSpacing 속성 없음

**에러 내용:**
```
error MC4005: 'System.Windows.Controls.TextBlock' 형식에서 Style Property 'CharacterSpacing'을(를) 찾을 수 없습니다.
```

**원인:**
- CSS의 `letter-spacing: 2px` 속성을 WPF `TextBlock.CharacterSpacing`으로 변환 시도
- WPF `TextBlock`은 `CharacterSpacing` 속성을 지원하지 않음

**수정 방법:**
- `CharacterSpacing` 설정 코드 제거
- 주석으로 WPF 제한사항 명시

**관련 파일:** `WetQuail86.Wpf.UI/Themes/WetQuail86.xaml:50`

---

### 에러 2: StackPanel.Spacing 속성 없음

**에러 내용:**
```
error MC3072: XML 네임스페이스 'http://schemas.microsoft.com/winfx/2006/xaml/presentation'에 'Spacing' 속성이 없습니다.
```

**원인:**
- WPF `StackPanel`은 `Spacing` 속성을 지원하지 않음 (AvaloniaUI 전용)

**수정 방법:**
- `Spacing="30"` 제거
- 각 자식 요소에 `Margin="0,0,0,30"` 적용

**관련 파일:** `WetQuail86.Wpf.Gallery/MainWindow.xaml:13`

## 잠재적 런타임 오류 (직접 확인 필요)

### 1. TextBox 포커스 동작
- `ScrollViewer` 내부의 `TextBox`가 포커스를 올바르게 받는지 확인 필요
- 키보드 입력이 제대로 전달되는지 테스트 필요

### 2. Opacity 애니메이션 충돌 가능성
- `IsMouseOver`와 `IsKeyboardFocusWithin` 트리거가 동시에 발생할 때 Opacity 애니메이션 충돌 가능
- 두 상태가 동시에 활성화될 경우 의도치 않은 동작 발생 가능

### 3. Title 툴팁 위치
- `Margin="0,40,0,0"`으로 고정된 Input Box 위치가 다양한 Title 길이에서 올바르게 표시되는지 확인 필요

## CSS → WPF 변환 매핑

| CSS 속성 | WPF 속성 | 변환 상태 |
|---------|---------|----------|
| `letter-spacing: 2px` | - | 미지원 (제거됨) |
| `transition: .2s ease-in-out` | `DoubleAnimation Duration="0:0:0.2"` | 변환됨 |
| `opacity: 0.8` | `Opacity="0.8"` | 변환됨 |
| `border-radius: 8px` | `CornerRadius="8"` | 변환됨 |
| `transform: rotate(45deg)` | `RotateTransform Angle="45"` | 변환됨 |
| `::before` pseudo-element | 별도 Border 요소 | 변환됨 |
