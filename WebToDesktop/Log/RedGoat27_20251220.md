## 잠재적 런타임 에러

### 1. MouseEnter 이벤트 트리거 중첩 실행
- **위치**: `RedGoat27.xaml` - EventTrigger (MouseEnter)
- **설명**: 마우스를 빠르게 여러 번 진입/이탈하면 스핀 애니메이션이 중첩 실행될 수 있음
- **권장 해결**: `BeginStoryboard.HandoffBehavior="SnapshotAndReplace"` 속성 추가 고려

### 2. GeometryGroup 렌더링 성능
- **위치**: `RedGoat27Resources.xaml` - CombinedGeometry
- **설명**: 6개의 PathGeometry를 GeometryGroup으로 결합하여 사용. 대량 렌더링 시 성능 저하 가능
- **권장 해결**: 성능 이슈 발생 시 단일 Path 문자열로 병합 고려

### 3. TransformGroup 순서 의존성
- **위치**: `RedGoat27.xaml` - TrophyViewbox.RenderTransform
- **설명**: TranslateTransform과 RotateTransform 순서에 따라 애니메이션 동작이 달라질 수 있음
- **현재 상태**: Translate → Rotate 순서로 정상 동작 예상

## CSS → WPF 변환 매핑

|----------|----------|
| `fill: gold` | `SolidColorBrush Color="Gold"` |
| `color: white` | `SolidColorBrush Color="White"` |

| `animation: bounce 0.5s` | `DoubleAnimationUsingKeyFrames Duration="0:0:0.5"` |
