## Runtime Error 가능성
다음 항목은 런타임에서 직접 확인이 필요합니다:

1. **Typography.Capitals 속성**
   - CSS `text-transform: uppercase`를 `Typography.Capitals="AllSmallCaps"`로 변환
   - 일부 폰트에서 AllSmallCaps가 지원되지 않을 수 있음
   - 대안: TextBlock에서 직접 대문자 텍스트 사용 또는 Converter 적용

2. **Path Geometry 렌더링**
   - 원본 SVG의 viewBox가 330x307 (Quote Icon)과 24x24 (Heart Icon)
   - WPF Path의 Stretch="Uniform"으로 크기 조정
   - 복잡한 Path 데이터가 정확히 렌더링되는지 확인 필요

3. **호버 애니메이션**
   - CSS `transition: 0.5s`를 WPF Storyboard로 변환
   - Trigger.EnterActions/ExitActions로 구현
   - 빠른 마우스 이동 시 애니메이션 중복 발생 가능
