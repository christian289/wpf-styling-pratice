## 잠재적 런타임 에러 가능성

### 1. 글로우 애니메이션 경로 문제
- **위치**: `GoodQuail97.xaml` - Style.Triggers EventTrigger
- **설명**: `(Control.Effect).(DropShadowEffect.Color)` 경로로 Effect에 접근하는데, ControlTemplate 내부의 Effect가 아닌 Control의 Effect를 참조함
- **영향**: 글로우 색상 애니메이션이 작동하지 않을 수 있음
- **해결 방안**: EventTrigger를 ControlTemplate.Triggers로 이동하고 TargetName 지정 필요
- **확인 필요**: 런타임에서 애니메이션 동작 확인 필요

### 2. ToggleButton 라디오 그룹 동작
- **위치**: `GoodQuail97.cs` - ToggleButton 상속
- **설명**: 현재 ToggleButton을 상속하므로 각 버튼이 독립적으로 토글됨 (라디오 버튼처럼 하나만 선택되지 않음)
- **영향**: 원본 HTML의 라디오 버튼 동작과 다름
- **해결 방안**: RadioButton을 상속하거나 GroupName 기능 추가 필요
- **확인 필요**: 사용자 요구사항에 따라 동작 확인 필요

## CSS → WPF 변환 매핑

| CSS 속성 | WPF 구현 |
|---------|---------|
| `width: 30px; height: 30px` | `Width="30" Height="30"` |
| `border-radius: 50px` | `CornerRadius="15"` |
| `background-color: rgba(5, 19, 16, 0.3)` | `SolidColorBrush Color="#4D051310"` |
| `box-shadow: 5px 5px 20px rgb(...)` | `DropShadowEffect BlurRadius="20"` |
| `transform: scale(0.9)` | `ScaleTransform ScaleX/ScaleY="0.9"` |
| `@keyframes glow` | `ColorAnimationUsingKeyFrames` + `RepeatBehavior="Forever"` |
| `transition: 0.5s` | `Duration="0:0:0.5"` |
| `::after` (checked) | `Ellipse` with Opacity animation |
