## 컴파일 에러 및 수정

### Error 1: XML 주석 내 `--` 문자열 사용 오류

**에러 메시지:**
```
error MC3000: 'An XML comment cannot contain '--', and '-' cannot be the last character. Line 5, position 38.' XML이 잘못되었습니다.
```

**원인:**
- `UglyPug52Resources.xaml` 파일에서 CSS 변수 매핑 주석에 `--name` 형식 사용
- XML 주석 `<!-- -->` 내부에 `--` 문자열 포함 불가

**수정 전:**
```xml
<!-- CSS Variables Mapping (CSS: --name)
     [width] = 100
     ...
-->
```

**수정 후:**
```xml
<!--
    CSS Variables Mapping
    [width] = 100
    ...
-->
```

**수정 방법:**
- `(CSS: --name)` 텍스트 제거
- XML 주석 규격에 맞게 `--` 문자열 회피

## 잠재적 Runtime Error 가능성

### 1. Tooltip 위치 계산
- 현재 고정된 Y 오프셋 값 사용 (`-53`, `-63` 등)
- 버튼 높이가 변경될 경우 Tooltip 위치가 맞지 않을 수 있음
- 해결 방안: 바인딩 + Converter를 사용한 동적 계산 고려

### 2. ClipToBounds 동작
- `Grid ClipToBounds="True"` 사용으로 텍스트/아이콘 슬라이드 효과 구현
- Tooltip은 Grid 외부에 있어 클리핑되지 않음 (의도된 동작)

### 3. 애니메이션 타이밍
- CSS `transition: 0.5s`를 WPF `Duration="0:0:0.5"`로 변환
- `CubicEase EasingMode="EaseOut"` 적용으로 CSS `ease` 유사 효과 구현
