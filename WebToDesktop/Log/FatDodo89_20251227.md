## 컴파일 에러 및 수정

### 에러 1: StackPanel Spacing 속성 없음

**에러 내용:**
```
MainWindow.xaml(11,77): error MC3072: XML 네임스페이스 'http://schemas.microsoft.com/winfx/2006/xaml/presentation'에 'Spacing' 속성이 없습니다.
```

**원인:**
- WPF의 StackPanel은 Spacing 속성을 지원하지 않음 (AvaloniaUI에서는 지원)
- WPF 제한사항 참조: html-css-to-wpf-xaml skill의 wpf-limitations.md#c009

**수정 방법:**
- StackPanel에서 `Spacing="20"` 제거
- 각 자식 요소에 `Margin="0,0,0,20"` 추가하여 간격 구현

```xml
<!-- 수정 전 -->
<StackPanel Spacing="20">
    <TextBlock ... />
    <controls:FatDodo89 ... />
</StackPanel>

<!-- 수정 후 -->
<StackPanel>
    <TextBlock ... Margin="0,0,0,20"/>
    <controls:FatDodo89 ... Margin="0,0,0,20"/>
</StackPanel>
```

## 잠재적 Runtime 오류 (직접 확인 필요)

### 1. SVG stroke-dasharray 애니메이션 미지원

**문제:**
- 원본 CSS는 stroke-dasharray 속성을 애니메이션하여 입 모양이 그려지는 효과 구현
- WPF Path의 StrokeDashArray는 DoubleCollection 타입으로 직접 애니메이션 불가
- 현재 구현은 StrokeDashOffset과 Rotation으로 대체하여 유사한 효과 구현

**영향:**
- 입 그려지는 효과가 원본과 약간 다를 수 있음
- 시각적 차이 확인 필요

### 2. 복잡한 Transform 체인

**문제:**
- 원본 CSS eye2 애니메이션: `rotate(40deg) translate(0, -56px) rotate(-40deg) scale(1)`
- WPF에서 이런 복잡한 Transform 체인을 정확히 재현하기 어려움
- RenderTransformOrigin으로 근사 구현

**영향:**
- 눈의 회전 궤적이 원본과 약간 다를 수 있음

### 3. LinearGradientBrush OpacityMask

**문제:**
- 원본 SVG는 mask와 linearGradient로 파란색 레이어에 그라데이션 적용
- WPF Canvas.OpacityMask로 구현했으나 정확한 블렌딩 효과 검증 필요

**영향:**
- 두 색상(시안, 파랑) 블렌딩이 원본과 다를 수 있음

## CSS → WPF 변환 매핑

| CSS                               | WPF                                              |
| --------------------------------- | ------------------------------------------------ |
| SVG viewBox="0 0 128 128"         | Viewbox + Canvas Width/Height="128"              |
| SVG circle                        | Ellipse                                          |
| SVG path (arc)                    | Path + EllipseGeometry                           |
| stroke-dasharray                  | StrokeDashArray                                  |
| stroke-linecap: round             | StrokeStartLineCap/StrokeEndLineCap="Round"      |
| transform-origin: 64px 64px       | RenderTransformOrigin="0.5,4.5" (상대 좌표 계산) |
| animation: 3s ease-in-out infinite| Storyboard RepeatBehavior="Forever"              |
| visibility: hidden                | Visibility="Hidden"                              |
| linearGradient + mask             | Canvas.OpacityMask + LinearGradientBrush         |
