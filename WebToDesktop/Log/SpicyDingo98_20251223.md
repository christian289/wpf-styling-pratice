## Runtime Error 가능성 (직접 확인 필요)

### 1. ItemContainerGenerator 타이밍 이슈
- **위치**: `SpicyDingo98Item.cs:OnIsSelectedChanged` 메서드
- **설명**: `ItemContainerGenerator.ContainerFromItem()`이 아직 컨테이너가 생성되지 않은 상태에서 `null`을 반환할 수 있음
- **영향**: 초기 선택 상태가 제대로 동기화되지 않을 수 있음
- **해결책**: `ItemContainerGenerator.Status`를 확인하거나 `StatusChanged` 이벤트 활용

### 2. 애니메이션 중첩
- **위치**: `SpicyDingo98.xaml` IsSelected Trigger
- **설명**: 빠르게 여러 아이템을 클릭할 경우 Storyboard가 중첩될 수 있음
- **영향**: 시각적 글리치 발생 가능
- **해결책**: `BeginStoryboard.HandoffBehavior="SnapshotAndReplace"` 적용 고려

### 3. DropShadowEffect 성능
- **위치**: `SpicyDingo98Resources.xaml` DropShadowEffect
- **설명**: 많은 아이템에 DropShadowEffect 적용 시 렌더링 성능 저하 가능
- **영향**: 다수 아이템 사용 시 UI 버벅임
- **해결책**: 아이템 수가 많을 경우 Effect 비활성화 또는 CacheMode 적용
