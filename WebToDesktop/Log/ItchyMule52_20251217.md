# ItchyMule52 변환 로그 (2025-12-17)

## 에러 내용 및 수정

### 1. StackPanel Spacing 속성 에러

**에러 메시지:**
```
MainWindow.xaml(14,21): error MC3072: XML 네임스페이스 'http://schemas.microsoft.com/winfx/2006/xaml/presentation'에 'Spacing' 속성이 없습니다.
```

**원인:**
WPF의 `StackPanel`은 `Spacing` 속성을 지원하지 않음 (AvaloniaUI에서는 지원)

**수정 방법:**
`Spacing` 속성 대신 각 자식 요소에 `Margin` 속성 사용

```xml
<!-- 수정 전 -->
<StackPanel Spacing="30">
    <controls:ItchyMule52/>
</StackPanel>

<!-- 수정 후 -->
<StackPanel>
    <controls:ItchyMule52 Margin="0,0,0,30"/>
</StackPanel>
```

## 잠재적 런타임 에러 가능성

### 1. 초기 상태에서 IsChecked="True"인 경우

**가능성:** 낮음

**설명:**
`IsChecked="True"`로 초기화된 컨트롤은 `Trigger.EnterActions`가 로드 시 실행되지 않을 수 있음.
따라서 초기 상태에서 Thumb 위치가 올바르지 않을 수 있음.

**확인 필요:**
런타임에서 `IsChecked="True"`로 초기화된 컨트롤의 Thumb 위치 확인 필요

### 2. Canvas 크기와 Thumb 이동 거리

**가능성:** 낮음

**설명:**
Thumb 이동 거리가 하드코딩(`To="28"`)되어 있어, 컨트롤 크기가 변경되면 비율이 맞지 않을 수 있음.

**권장 사항:**
동적 크기 조절이 필요한 경우 Converter를 사용하여 이동 거리 계산 필요
