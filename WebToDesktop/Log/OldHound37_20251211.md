## 컴파일 에러 및 수정 내용

### 에러 1: XML 주석 내 이중 대시 문자

**에러 메시지:**
```
OldHound37Resources.xaml(28,33): error MC3000: 'An XML comment cannot contain '--', and '-' cannot be the last character.'
```

**원인:**
- XML 주석 내에 CSS 변수명 `--s`를 그대로 사용
- XML 표준에서 주석 내 `--` 문자열은 허용되지 않음

**수정 전:**
```xml
<!-- Default Tile Size (CSS --s: 200px) -->
```

**수정 후:**
```xml
<!-- Default Tile Size (CSS variable s: 200px) -->
```

## 잠재적 Runtime Error

### 1. DrawingBrush 타일 패턴 시각적 차이
- **설명:** CSS `conic-gradient`는 WPF에서 직접 지원되지 않음
- **구현 방법:** DrawingBrush + GeometryDrawing으로 근사 구현
- **영향:** 원본 CSS 패턴과 시각적 차이가 있을 수 있음 (특히 골드 그라데이션 영역)
- **확인 필요:** 실제 실행하여 시각적 결과 확인 필요

### 2. TileSize DependencyProperty 미적용
- **설명:** `TileSize` 속성이 정의되어 있으나 XAML 템플릿에서 하드코딩된 값 사용
- **현재 상태:** DrawingBrush Viewport가 `200,200`으로 고정
- **영향:** TileSize 속성 변경 시 실제 타일 크기가 변하지 않음
- **해결 방안:** TemplateBinding 또는 Converter를 사용하여 동적 바인딩 필요

### 3. 대각선 라인 렌더링 품질
- **설명:** LineGeometry로 구현한 대각선 라인은 안티앨리어싱에 따라 품질이 달라질 수 있음
- **확인 필요:** 다양한 해상도 및 DPI 설정에서 테스트 필요

## CSS → WPF 변환 주요 매핑

| CSS 속성 | WPF 구현 |
|---------|---------|
| `conic-gradient` | DrawingBrush + GeometryDrawing (근사 구현) |
| `repeating-linear-gradient` | LineGeometry + Pen |
| `repeating-conic-gradient` | RectangleGeometry 체커보드 패턴 |
| `background-size` | DrawingBrush Viewport/Viewbox |
| CSS 변수 (`--s`) | DependencyProperty + StaticResource |
